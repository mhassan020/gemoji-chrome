gemoji-chrome
=============

Chrome plugin for finding and inserting gemoji shortcodes.

Link to the plugin on the [chrome web-store](https://chrome.google.com/webstore/detail/gemoji-chrome/ngbmhomembgkmpjkgppjmofejjejonkb).
